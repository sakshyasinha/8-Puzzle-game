# 8-Puzzle-game

A classic **8-Puzzle** (sliding puzzle) implementation where the goal is to arrange the tiles from **1–8** in order by sliding them into the empty space.

## Features
- 3×3 sliding puzzle (tiles 1–8 + one blank)
- Shuffle / randomized start state
- Move tiles by sliding into the empty space
- Detects win condition

## How to Play
1. Start the game.
2. Slide a tile into the empty space (usually by clicking the tile or using keyboard controls—depending on the implementation).
3. Continue until the board matches the goal state:

``
1 2 3
4 5 6
7 8 _
``
